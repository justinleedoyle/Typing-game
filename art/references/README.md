# Reference images

Drop the image files here using the filenames listed in `../PROMPTS.md` ("Reference image checklist" section).

These are the style anchors uploaded into image generators alongside the text prompts. Keep them in PNG (or JPG/WEBP if size matters) — they get committed to the repo so the art direction is reproducible across sessions and machines.

The character sheet (`wren-character-sheet.png`) is the master anchor and should be uploaded as a style reference for *every* generation, not just Wren-specific ones.
